Do you like Soutaisei Riron and Yakushimaru Etsuko's vocal? Okay, me too. So I wrote a small software, yksmrtime. It is a Yakushimaru Etsuko version of saytime. Or, a command line port of Tokei Tick Endless.

Of course, it is written in Scala. You can install from conscript. 

% cs kzys/yksmrtime

Conscripted kzys/yksmrtime to /Users/kzys/bin/yksmrtime
% yksmrtime
%

Enjoy!
